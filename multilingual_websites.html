Overview

So, what is this multilingual thing anyway? Multilinugal, or multiple language, websites are sites that can be viewed in more than one language. This plugin makes creating a multilingual site a breeze through the use of a new 'lang' element.

Step 3:

Create a Language Folder somewhere on your website's filesystem. This is where you will put all of your language files.


Step 4:

Create language files in this folder using ll-cc naming convention (ll = Language Subtag, cc = Country Subtag). For the official Language and Country Subtag Registry go to: http://www.iana.org/assignments/language-subtag-registry/language-subtag-registry or the unofficial searchable Registry: https://r12a.github.io/apps/subtags/.
The W3 spec can be found here.
Call the file ll-cc.lang, replacing ll with the Language Subtag and the cc with the Country Subtag as detailed above.


Step 5:

Set your HTML page's default language in the html element so it looks like:

<html lang="ll-cc">


Step 6:

Activate tfff1's multilingual websites by inserting the following in to the header of your web page (after JQuery and multilingual.js, replacing "To-Language-Folder" with the location of your Language Folder from Step 4):

                    
<script type="text/javascript>
  $(window).ready(function() {
    var attribute = $('html').attr('lang');
    loadLanguage(attribute, "To-Language-Folder");
  }); 
</script>

Creating a language-changing element

There are two ways to create a language-changing element, one is easily scalable and one isn't:

First Way (Recommended, Easily Scalable)
Second Way (Slightly Easier, Not Scalable)

First Way - External Language Options

With this way languages are saved in external files named (as mentioned in Step 4) ll-cc.lang


NOTE: In this tutorial I will be using en-us and fr-fr as my two languages. In use, just change these to what ever language/country you like, the same idea still applies.

The Mission

To recreate the code below in both English(United States) & French(France).

                    
<h1>Hi</h1>
<div><a href="www.example.com">Example Website</a></div>

Method

Step 1:

Create two files called "en-us.lang" & "fr-fr.lang" in my languages Directory.


Step 2:

In my "en-us.lang" file I will add:


header=Hi
example_site=Example Website

In my "fr-fr.lang" file I will add:

header=Salut
example_site=Exemple Site


Step 2:

I will now change the HTML to look like this:

                    
<h1><lang langID="header"></lang></h1>
<div><a href="www.example.com"><lang langID="example_site"></lang></a></div>
Now if I change the language to French it will look like: 
                    
<h1>Salut</h1>
<div><a href="www.example.com">Exemple Site</a></div>
And in English it will look like:
                    
<h1>Hi</h1>
<div><a href="www.example.com">Example Website</a></div>

This way works best and is most easily scalable to any number of different languages.

Second Way - Internal Language Options

This way embeds the language information in the HTML tag of the lang element it affects. For Example, the above example, when finished, would look like: 

                    
<h1><lang en-us="Hi" fr-fr="Salut"></lang></h1>
<div><a href="www.example.com"><lang en-us="Example Website" fr-fr="Exemple Site"></lang></a></div>

This would dramatically increase download times for your website, as instead of downloading one language's worth of data, the user will be downloading the data for all of the language your website supports.


Switching Languages

Switching languages is very easy. Simple use the following javascript code: (ll = Language, cc = Country)

                    
changeLanguage('ll-cc');

Same language accross pages

Keeping the language the same across all of your pages is remarkably easy. Simple execute a javascript function similar to that below:

                    
location.href=langNavigate('/new_page.html');

This will return "/new_page?lang=ll-cc" (ll = Language, cc = Country)


Setting Language from URL

You can easily set the page's language from the URL. Simple add ?lang= ll-cc (ll = Language, cc = Country) as demonstrated below:

                    
http://www.example.com/?lang=en-us

This will display the page in en-us.
